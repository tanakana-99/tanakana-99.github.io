自己紹介

Webコーダー Tanakaです。インターネットアカデミーにてWeb知識を学び、HTML/CSS/JavaScriptを使用したウェブサイト制作を行っています。

私にできること

Webサイトの設計からコーディングまで、一貫して丁寧に対応いたします。デザインカンプの正確な読み取りをもとに、意図を汲み取ったコーディングを得意としています。

HTML / CSS

レスポンシブ対応、SEOを意識した正確なコーディングが可能です。

JavaScript / jQuery

アニメーションや動的コンテンツの実装が得意です。

Photoshop / Adobe XD / Figma

デザインツールの操作に慣れており、コーディングに必要な情報の正確な読み取りと素材の書き出しが可能です。
